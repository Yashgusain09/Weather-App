# 🌦️ Weather App

A responsive Weather Web Application built using **HTML, CSS, and JavaScript** that fetches real-time weather information using the **OpenWeather API**.

## 📌 Features

* 🔍 Search weather by city name
* 🌡️ Displays current temperature
* ☁️ Shows weather condition
* 💧 Displays humidity
* 🌬️ Shows wind speed
* 📊 Displays atmospheric pressure
* 🤗 Shows "Feels Like" temperature
* 🖼️ Dynamic full-screen background based on weather conditions
* 📱 Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* OpenWeather API

## 🚀 How to Run the Project

1. Clone this repository.

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

3. Get a free API key from **OpenWeather**.

4. Replace the following line in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your own API key.

5. Open `index.html` in your browser or run the project using **Live Server** in Visual Studio Code.
